# CarsPlatform

## Overview
This university project, built using ASP.NET Core, focuses on managing car data efficiently. The application reads data from a CSV file and seamlessly inserts it into a database. It also provides full CRUD (Create, Read, Update, Delete) functionality, enabling users to manage car information effectively.
